Remote Notes Server

Remote Speaker notes for Reveal.js

What is it for?

For presentations with speaker notes, that can be used on addional/remote devices.

How does it work?

The presentation is uploaded as a ZIP to the notes-server via upload-presentation.

The server serves the slides plus the associated notes available in the browser.

It is currently not ment as an addional presentation provider.

Setup


                       # start the notes server
                       # this uses ./presentations as storage
                       ./notes-server --port=1947
                       # upload the slides to the server
                       ./upload-presentation --server-url http://localhost:1947 \
                                             --html-file notes-client/demo.html

The next slide should indicate, a connection.

Get started

Integrate remote notes


<script>
window.REMOTE_NOTES_CLIENT_CONFIG = {
  serverUrl: "http://localhost:1947",
  socketId: "my-talk",
  socketIoPath: "./presentation-libs/socket.io.js",
  reveal: window.Reveal,
  revealConfig: {
    plugins: [RevealHighlight, RevealNotes],
  },
};
</script>
<script src="./remote-notes-client.js"></script>
<script>window.RemoteNotesClient.init();</script>

socketId is the presentation name (filename of uploaded presentation without extension)

use upload-presentation --name to set it manually
